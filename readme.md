# Python Quiz Game

## Project Overview

The Python Quiz Game is a simple command-line application developed using Python. It asks multiple-choice questions, accepts user input, checks the answers, calculates the final score, and displays the performance at the end.

## Features

- Multiple-choice quiz
- User-friendly interface
- Input validation (A, B, C, D)
- Automatic score calculation
- Percentage calculation
- Performance evaluation
- Displays correct answers at the end

## Technologies Used

- Python 3
- Visual Studio Code

## How to Run

1. Install Python 3.
2. Open the project in Visual Studio Code.
3. Open the terminal.
4. Run the following command:

```
python quiz_game.py
```

5. Answer the questions by entering A, B, C, or D.
